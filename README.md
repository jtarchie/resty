# Resty

Resty is a web framework that forces restful conventions that are inspired by Rails Responders.

## Installation

Add this line to your application's Gemfile:

    gem 'resty', github: "jtarchie/resty"

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install resty
